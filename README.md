React Generator
===============

## 说明 ##
	移动端React项目模板，用于开发移动端React项目
	自带20+常用组件，支持rem屏幕适配
	version:webpack(2.4.5) react-router(4)

## 技术栈 ##
	React React-Router Redux Sass PostCss Webpack iscroll ES6 Rem Node(6.10.3)

## debug ##

## mock ##
